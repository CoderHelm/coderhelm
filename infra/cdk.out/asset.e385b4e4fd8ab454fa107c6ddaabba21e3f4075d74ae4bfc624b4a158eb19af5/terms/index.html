2. Description of Service

Coderhelm is an autonomous AI coding agent delivered as a GitHub App. When you assign a GitHub issue to Coderhelm or apply a designated label, the Service:

  • Reads your repository files and structure to understand context
  • Analyzes the issue and generates a plan for code changes
  • Implements the changes across one or more files
  • Self-reviews the changes through multiple automated passes
  • Opens a draft pull request for your review
  • Optionally detects and fixes CI failures on its branch

The Service also includes:

  • Jira integration: You may connect a Jira workspace to sync issues and trigger runs from Jira tickets
  • Google OAuth login: You may authenticate to the dashboard using your Google account
